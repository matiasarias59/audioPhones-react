# Audio Phones Web 
Audio Phones Web es un catalogo online de productos electronicos. 

Aplicación Web desarrollada con ReactJs

Deploy: https://audio-phones-reactjs.netlify.app/#/

## Descripcion de la Aplicación

Audio Phones Web es un projecto que surge de la necesidad de mostrar un catalogo online actualizado a tiempo real a patir de datos almacenados en una hoja de cálculo de Google.



## Librerias utilizadas

    React: V18.2.0,
    React Router Dom: V6.6.1
